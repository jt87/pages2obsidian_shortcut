The Shortcut does the following: prompts you to select one or multiple .pages documents, prompts you to choose the export location, exports .pages to .docx in the background, and then runs a Shell Script which converts the .docx to .md with Pandoc while tailoring the markdown format for Obsidian.

Check the text doc if you'd like to examine the Shell Script.

Note that a prerequisite for using this Shortcut is having Pandoc installed on your Mac. If you don't have it, you'll be given an error with directions on how to install it with terminal.
